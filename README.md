🌿 HydroJungle Laundry Services

HydroJungle Laundry Services is a full-stack web application built using Django and React, designed to modernize and simplify laundry service management. The platform allows users to schedule pickups, track orders, and manage services through a smooth, interactive interface enhanced with rich animations.

🚀 Features
🧺 Online laundry service booking
📅 Pickup & delivery scheduling
🔍 Real-time order tracking
👤 User authentication & management
🎨 Smooth UI animations and responsive design
⚡ Fast and interactive frontend using React
🔒 Secure backend powered by Django
